# GeoJSON endpoints for Japanese administrations by 国勢調査境界データ

```
https://geolonia.github.io/japanese-admins-kokuseichosa/<prefCode>/<adminCode>.json
```

## ビルド方法

```
$ ./bin/download.sh
```
